Contexte

Le
projet

Ce site web a été réalisé dans le cadre du projet de programmation Web TI402 (2025–2026), encadré par CHIBOUT KHELIFA Lydia à l'EFREI Paris Panthéon-Assas Université.

L'objectif était de concevoir et développer un site vitrine du Département d'Informatique de l'EFREI, en utilisant uniquement HTML5, CSS3 et JavaScript — sans frameworks ni bibliothèques externes.

HTML5 CSS3 JavaScript Vanilla Responsive Design W3C Valide
